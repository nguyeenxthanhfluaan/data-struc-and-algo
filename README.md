# Data Structure And Algo

## :computer: How to run

### 1. Clone the repository:

```
$ git clone https://github.com/buivantann/data-struc-and-algo.git
```

### 2. Install package

```
$ npm run init
```

### 3. Create file .env in server folder, and set MONGO_URI to .env file

### 4. Run

```
$ npm run dev
```
