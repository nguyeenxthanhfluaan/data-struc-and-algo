# Data Structure Search

## :computer: How to run

### 1. Clone the repository:

```
$ git clone https://github.com/buivantann/data-structures-and-algorithm.git
```

### 2. Install package

```
$ npm run init
```

### 3. Add file .env to server folder, and supply MONGO_URI to .env file

### 4. Run

```
$ npm run dev
```
